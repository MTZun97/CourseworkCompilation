Hypothesis testing result:¶

Although no clear patterns are evident in Fig.1, a two-sample test was conducted between the lowest and highest qualities (Quality=3 vs. Quality=8) for the Acidity variable. Results show that, while high-quality wines had a slightly greater mean acidity than the low-quality wines, this effect failed to reach significance at $\alpha$=0.05 (t = 0.261, p=0.796) due to relatively large aciditiy variance (Fig.2).

Machine learning result:¶

Similar to the hypothesis testing results above, a preliminary classification analysis was conducted using low- and high-quality wines (quality=3 and quality=8, respectively). A Multi-Layer Perceptron (MLP) classifier was able to classify these wines with a classification rate of 0.893 (Fig.3). These results support the hypothesis testing interpretation above, that separating high- from low-quality wines requires multiple variables. In particular, the results showed that high acidity and low citric was generally associated with low-quality wines (Fig.3).

def plot_decision_surface(classifier, x, labels, ax=None, colors=None, n=50, alpha=0.3, marker_size=200, marker_alpha=0.9):
    nlabels   = np.unique( labels ).size
    colors    = plt.cm.viridis( np.linspace(0,1,nlabels) )  if (colors is None) else colors
    ax        = plt.gca() if (ax is None) else ax
    xmin,xmax = x.min(axis=0), x.max(axis=0)
    Xp,Yp     = np.meshgrid( np.linspace(xmin[0],xmax[0],n) , np.linspace(xmin[1],xmax[1],n) )
    xp        = np.vstack( [Xp.flatten(), Yp.flatten()] ).T
    labelsp   = classifier.predict(xp)
    Labelsp   = np.reshape(labelsp, Xp.shape)
    cmap      = ListedColormap(colors)
    for i,label in enumerate( np.unique(labels) ):
        xx   = x[labels==label]
        ax.scatter( xx[:,0], xx[:,1], color=colors[i], s=marker_size, alpha=marker_alpha, label=f'Quality = {label}' )
    plt.pcolormesh(Xp, Yp, Labelsp, cmap=cmap, alpha=alpha, shading='nearest')
    ax.set_xlabel('Acidity')
    ax.set_ylabel('Citric')
    ax.legend()
